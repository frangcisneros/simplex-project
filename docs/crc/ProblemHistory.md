# Tarjeta CRC: ProblemHistory

### Clase: ProblemHistory

**Responsabilidades:**

- Gestionar historial de problemas resueltos previamente
- Listar problemas del historial con información resumida
- Recuperar un problema específico del historial por ID
- Mostrar información detallada de un problema (solución, tiempo, iteraciones)
- Eliminar problemas del historial
- Verificar si existen problemas en el historial
- Proporcionar interfaz interactiva para navegar el historial
- Formatear y colorear output en consola
- Integrar con LoggingSystem para consultar base de datos

**Colaboradores:**

- `LoggingSystem` - Accede a base de datos de historial
- `Colors` - Clase de utilidad para colores en consola

**Ubicación:** `simplex_solver/problem_history.py`

**Tipo:** Gestor de historial (History Manager)
